# bradleyhighclassfund.com

Educational, informational website showing a student-selected equity portfolio with delayed quotes.

## Local dev
1. Install Node.js 18+
2. `npm install`
3. `npm run dev`

## Vercel environment variables
- NEXT_PUBLIC_BASE_URL = https://bradleyhighclassfund.com (or your Vercel preview URL while testing)
- API_NINJAS_KEY = <your key> (required for refresh)

## Deploy
Push to GitHub and import into Vercel.
